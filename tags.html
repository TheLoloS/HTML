Formatowanie tekstu

 commment - <!---->
 <!DOCTYPE html> - mówi przeglądarce, jaki to rodzaj dokumentu.
 < a > - link
 < abbr > - okno kontekstowe.
 < address > - dane autora i inne (mało przydatne).
 <area shape="kształt" coords="koordynaty" href="link aktywowany po kliknięciu" alt="po najechaniu"> - tworzy strefy na zdjęciu, które są interaktywne.
 < article > - akapit
 < aside > - nie użyteczne
 < audio controls > - dodanie muzyki w środku tag source
 < b > pogrubia tekst
 < bdi > pomaga przy arabskim pisaniu od prawej strony
 < bdo > ułyt do tsket ezsip (w dir wybierasz, w która stronę)
 < big > html4 powiększenie tekstu
 < blockqiute > - dodaje cytat oraz źródło? Mało potrzebne
 < cite > - pisze się w nim tytuły oraz nazwy (dodaje czcionkę italic). Daje się go np. w p tagu
 < code > - zmienia czcionkę na bardziej "kodową"
 < del > - przekreśla tekst
 < dfn > - dodaje do tekstu okno dialogowe oraz zmienia czcionkę
 < em > - dodaje czcionkę italic
 < i > - czcionka italic. Piszą, że używać w ostateczności jak semantycznie nic innego nie pasuje
 < ins > - podkreśla tekst
 < kbd > - zmienia czcionkę. Służy do pisania o klawiszach klawiatury
 < mark > - zaznacza tekst kolorkiem
 < meter > - fajna rzecz. Robi pasek postępu w zależności, jakie wartości mu podasz
 < pre > - zmienia czcionkę na bardziej "kodową" oraz respektuje taby i entery
 < progress > - to samo co meter tylko używane do progresu xD tez fajne a inny kolorek
 < q > - dodaje apostrofy
 < rp rt ruby > - taka niby potęga to pomaga przy znakach azjatyckich
 < s > - przekreślenie
 < samp > - zmienia czcionkę na bardziej "kodową"
 < small > - zmniejsza czcionkę 
 < strong > - pogrubienie
 < sub / sup > - potęga i pierwiastek, wyżej niżej
 < template > - kozak rzecz! W nim umieszczasz rzeczy i strona na początku ich nie renderuje, dopiero za pomocą JS możesz ich użyć, a potem wykorzystywać je bez końca bez tworzenia elementów w JS
 < time > - generalnie nic nie robi i służy do umieszczania czasu
 < u > - podkreśla tekst
 < var > - nic nie robi, zmienia czcionkę...
 < wbr > - definiuje wcięcie w p
